# Attention Place 웹사이트

`attentionplace.com` 에 올릴 한 페이지 소개 사이트입니다.

## 파일 구성
- `index.html` — 메인 소개 페이지
- `privacy.html` — 개인정보처리방침 (App Store 제출 시 필요)
- `CNAME` — 커스텀 도메인(attentionplace.com) 연결용 파일 (GitHub Pages가 자동 인식)

## 미리보기 (내 컴퓨터에서)
`index.html` 파일을 더블클릭해 브라우저로 열면 됩니다.

## 배포 방법은 채팅의 단계별 안내를 참고하세요.
1. 도메인 구매 (가비아 또는 Cloudflare)
2. GitHub 저장소 → GitHub Pages 활성화
3. DNS 설정으로 도메인 연결
4. HTTPS 적용 확인
